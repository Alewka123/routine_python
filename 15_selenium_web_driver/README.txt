1. необходимые зависимости : 
-python 3.6+
-selenium  3.141.0+
-cromedriver 80+
2. настройка конфига 
- в файл конфига cred.cfg ввести данные
-- login - логин на ХХ
-- password - пароль на ХХ
-- resumeID - ИД резюме на ХХ (адрес открытой страницы с резюме)
-- browserPath - путь до вебдрайвера
-- reply - кол-во ответов на вакансии в режиме автомата
-- replyMessage - текст сопроводительного письма для ответа на вакансию
